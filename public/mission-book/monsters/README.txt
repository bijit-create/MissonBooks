Drop monster character PNGs here.

Filenames must match the `asset` paths used in src/missionBook/monsters.ts, e.g.:
  muncher.png
  crossangle.png
  problemeter.png
  traity.png

Recommended:
  - Square aspect (e.g. 512x512), transparent background.
  - Single character centered, with no surrounding plate or text.

If a PNG is missing for a mapped LO, the PDF falls back to the colored name-pill
only (no character image).
